# 外运物流控制塔设计系统

## 项目概述
外运物流控制塔是一个基于数字孪生、AI预测和可视化大屏的智能物流管理平台。整合三个核心能力：
1. **数字孪生可视化** (DigitalTwinScreen) - 实时3D物流场景监控
2. **时间序列预测** (TimesFM) - 物流需求预测与优化
3. **设计系统** (DESIGN.md) - 统一的UI/UX设计规范

## 设计原则

### 1. 实时性 (Real-time)
- 所有数据实时更新，延迟不超过5秒
- 3D场景与真实物流场景同步
- 预测结果实时刷新

### 2. 可视化优先 (Visualization First)
- 复杂数据通过3D可视化呈现
- 多维度数据融合展示
- 直观的交互式操作界面

### 3. 预测驱动 (Prediction Driven)
- 基于TimesFM的时间序列预测
- 多维度预测：需求、运力、成本、风险
- 预测结果可视化展示

### 4. 智能协同 (Intelligent Collaboration)
- 多部门协同工作台
- 智能预警与推荐
- 自动化决策支持

## 系统架构

### 前端架构
- **框架**: Vue 3 + TypeScript
- **3D引擎**: Three.js + Cesium.js (地理空间可视化)
- **图表**: ECharts 5 + D3.js
- **设计系统**: 基于DESIGN.md的组件库

### 后端架构
- **框架**: FastAPI (Python)
- **数据库**: PostgreSQL + TimescaleDB (时序数据)
- **缓存**: Redis
- **消息队列**: RabbitMQ
- **AI服务**: TimesFM预测服务

### 预测模块
- **模型**: TimesFM 2.5 (200M参数)
- **预测范围**: 需求预测、运力预测、成本预测、风险预测
- **数据源**: 历史订单、天气、经济指标、交通数据

## 数据流设计

### 实时数据流
```
传感器/系统 → Kafka → 流处理 → Redis → 前端展示
```

### 预测数据流
```
历史数据 → TimesFM → 预测结果 → 可视化展示
```

### 批处理数据流
```
业务数据 → ETL → 数据仓库 → 分析报表
```
