## 技术栈

### 前端
- **框架**: Vue 3 + TypeScript
- **3D引擎**: Three.js + Cesium.js
- **图表库**: ECharts 5 + D3.js
- **状态管理**: Pinia
- **构建工具**: Vite

### 后端
- **框架**: FastAPI (Python 3.9+)
- **数据库**: PostgreSQL + TimescaleDB
- **缓存**: Redis
- **消息队列**: RabbitMQ
- **AI服务**: TimesFM + PyTorch

### 部署
- **容器化**: Docker + Docker Compose
- **反向代理**: Nginx
- **监控**: Prometheus + Grafana
